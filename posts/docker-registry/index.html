Docker Registry

2019-04-01
#docker 

how to move a docker image to another registry (example push to localhost registry) #

first you need to tag an existing image to another name containing the target registry do a docker images to get a list of images already downloaded command below tags redis:3-alpine as localhost:5000/redis:3-alpine

docker tag redis:3-alpine localhost:5000/redis:3-alpine

then just push the new tag

docker push localhost:5000/redis:3-alpine

and with a curl you can see the output of the catalog in our 127.0.0.1 (localhost) registry #

curl http://127.0.0.1:5000/v2/_catalog

more info : https://stackoverflow.com/questions/31251356/how-to-get-a-list-of-images-on-docker-registry-v2
