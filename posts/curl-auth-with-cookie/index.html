curl auth with cookie

2018-04-18
#curl  #notes 

use curl to simulate authentication in apis that provide a cookie via auth endpoint (nimbula was using this)

curl -kc ./authtokens -X POST -H "Content-Type: text/plain" -d '{"result": null, "password": "testpassword", "uri": null, "user": "/test/user"}' https://<server>:445/authenticate/

then you can use the cookie from the file to get something:

curl -k -b ./authtokens -X GET "https://<server>:445/node/"
